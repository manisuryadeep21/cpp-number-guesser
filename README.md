# Guessing Game in C++
## 🚀 How to Run the Game Locally

### Prerequisites
Make sure you have a C++ compiler installed on your system (like `g++` via MinGW for Windows, or Clang for macOS/Linux).

### Compilation Steps
1. Clone this repository or download the source files.
2. Open your terminal or command prompt in the project folder.
3. Compile the code using the following command:
   ```bash
   g++ main.cpp -o Game
   ```

### Execution
Run the compiled game executable:
* **Windows:** `.\Game.exe`
* **macOS / Linux:** `./Game`
